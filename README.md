# RASCH CALCULATOR IN UZBEK LANGUAGE

## install
#### clone project
```shell
git clone https://github.com/HeliseiKroopin/rasch-calc.git
```

#### open project folder
```shell
cd rasch-calc
```

#### make project

```shell
make
```

#### run result file
```shell
./build/result
```
